# 종종 등장하는 비기술적 질문

## 만약 새로운 개발 업무가 주어졌을 때, 회사에서 그 누구도 그 개발 업무에 관한 지식이나 경험이 없어 가르쳐 줄 수 없을 대 어떻게 해결하는가?

인터넷을 아무리 뒤져도, 주변의 지인이 많아도 모르는 문제인 경우가 종종 있다. 문제 해결 능력이란 이럴 때 문제를 효과적으로 해결할 수 있는 능력이라고 생각한다.  
일단 문제를 분석하고 핵심을 파악해 핵심별로 해결책을 찾아 제시하고 실행할 수 있어야 한다고 생각한다.  
간혹 자신은 어떤 언어만 사용한다거나 어떤 기술만 사용해서 해결 할 수 없다고 하는 사람도 있지만, 특정 기술이나 언어에 종속적인 개발자는 그런 능력이 한정적인 거라 생각한다.  
사용해보지 않은 기술이나 언어로는 해결 할 수 있는지 찾아보고 API 명세서를 확인 해서 적절한 기술을 찾아 볼 것이다.
