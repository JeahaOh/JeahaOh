- Spring Framework

- MSA

- Node.js
- Java

"야핏무브 및 야핏사이클에서 사용할 플랫폼 및 공용 API 개발"
-  회원 및 회원의 서비스 데이터 관리 개발
-  공용 라이브러리 및 SDK 개발
-  모바일/웹 환경을 위한 API 개발

## 자격요건

- 총 개발 경력 8년 이상 (Java 혹은 Kotlin / Spring 기반 경력 필수)
- API 설계 및 구현이 가능히신 분
- 새로운 기술에 관심이 많고 도입에 적극적이신 분
- 유연하면서도 가독성이 좋은 코드를 작성하실 수 있는 분
